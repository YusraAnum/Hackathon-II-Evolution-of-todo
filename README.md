# Todo App - All Phases

This repository contains all three phases of the Todo application development:

## Phase 1: CLI Todo Application
Located in `phase-1/` directory
- In-memory CLI Todo with full SDD artifacts
- Core functionality with add, list, update, delete operations
- Includes tests and numeric ID support

## Phase 2: Full Stack Todo Application  
Located in `phase-2/` directory
- Backend with FastAPI and SQLModel
- Frontend with React/Next.js
- Complete CRUD operations for todos
- User authentication and management

## Phase 3: AI Todo Chatbot
Located in `phase-3/` directory
- AI-powered chatbot for todo management
- MCP tools for task operations
- OpenAI integration for natural language processing
- Complete frontend with chat interface
